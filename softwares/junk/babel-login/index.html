babel-login

1989年公開の押井守監督のアニメ映画「機動警察パトレイバー the Movie」に出てくる、篠原重工の帆場暎一が開発したレイバー用OS「HOS（Hyper Operating System）」の起動画面を模したパスワード認証プログラムです。
ジョークプログラム（危険なプログラムではありません）ではありますが、実際にログイン画面としても使えます。

起動はこちらから

上記リンクから起動できます。
パスワードは半角で「1999」です。途中で入力ダイアログが表示されるので入力してください。
原作同様、ログインに失敗すると画面に「BABEL」という文字が次々と表示され、延々と画面上を流れていきます。
ログインに成功すると、すえーでんのウェブサイトのトップ画面へと飛びます。
ログイン成功・失敗の 2 パターンとも体験してみると面白いかもしれません。

ネタが分かる人はぜひ楽しんでください。分からない人もぜひネットで検索。
劇場版機動警察パトレイバーは、1 も 2 も名作です。

ソースコードはこちら

ソースコードの閲覧は上記リンクから。すえーでんの GitHub の該当ページへと飛びます。

作者メモ

今回は勉強がてら、Canvas（Canvas API）を用いて実装しました。使用言語としては HTML5 と CSS、JavaScript です。
ログイン画面としても一応使えますが、今回は Canvas を用いた2Dアニメーションの実装の勉強を目的として作ったため、パスワード認証部分は JavaScript で文字列一致を検出するだけのものとなっています。
「セキュリティ的にちょっと…」という人は、データベースを用いるなどして適宜書き換えてください。
また完成させることを最優先に書いたので、正直ソースコードは汚いです。アレンジも作者の自己主張の強いものとなっています。
