youdown.el

Youtube Downloader

Youtube の動画を簡単にダウンロードするための Emacs Lisp スクリプトです。Windows上での使用を想定しています。

「ダウンローダー」と格好つけてはいますが、ダウンロード操作そのものは yt-dlp に任せているため、本スクリプトは yt-dlp を楽に扱うための「ラッパー」というのが実態です。

使用するには Emacs と yt-dlp のインストールが必要です。

ソースコードの閲覧およびスクリプトのダウンロードはこちらから

導入方法や使用方法については上記リンク先にて詳しく記述していますので、そちらを参考にして下さい。
