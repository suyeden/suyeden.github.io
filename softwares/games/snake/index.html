ランキング

おそらく誰もが一度は見たことあるゲームなのではないかと思います。
ヘビを操作して、頭が壁や自分自身にぶつからないようにしながらエサを食べて身体を伸ばし、画面をすべて埋めつくすことができればゲームクリアです。

ゲームを起動するとランキングへの参加意思を聞かれるため、参加する場合は「OK」を選択して10文字以内の名前を入力してください。

このゲームは HTML5、CSS、JavaScript を使用して、また Canvas を用いて実装しています。
さらに Ajax を使った Web アプリの実装を経験したかったため、これを使ってランキング機能を実装しました。
