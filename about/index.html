About

すえーでん（suyeden）

1999年2月12日ベルギー生まれ。大阪府在住。
大阪教育大学 教育学部 教育協働学科 理数情報専攻 自然科学コース 卒業。
高校時代に出会った友人の影響で Linux (ディストロは KNOPPIX) に触れたことがきっかけで、コンピュータにのめり込むようになりました。独学の趣味プログラマ程度の実力ではありますが、今も暇さえあれば何かしらのプログラムを組むなどして遊んでいます。

使用している開発環境は Emacs です。初めに触れた KNOPPIX に VSCode が入らなかったため仕方なく使っていたら、そのまま慣れてしまいました。
普段は Windows 使いですが、自宅では Debian GNU/Linux も使っていて、気分によって使い分けています。
好きな言語は Emacs Lisp で、日常的に使うツールや書き捨てのスクリプトのようなちょっとしたものを作るときに、シェルスクリプト代わりによく使っています。
他に扱える言語として、HTML5・CSS・JavaScript・PHP は本サイトの構築のために使用しました。C や C++ はそこまで経験が深いわけではありませんが、競技プログラミングや Emacs の拡張モジュールを書く時などにたまに使います。Python は大学の講義で文法を学んだ程度で、Bash や PowerShell/cmd はコマンド操作やシェルスクリプト/バッチファイルの記述のためによく使用しています。
その他スキルとしては Git/GitHub・MySQL も扱っています。
