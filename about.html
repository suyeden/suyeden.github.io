About

すえーでん（suyeden）

1999年2月12日ベルギー生まれ。大阪府在住。
大阪教育大学 教育学部 教育協働学科 理数情報専攻 自然科学コース 在籍。
高校時代に出会った友人の影響で Linux (ディストロは KNOPPIX) に触れたのがすべての始まり。
以来のんびり独学しながら今日に至る。

使用している開発環境は Emacs と Visual Studio Code。好きな言語は Emacs Lisp。
普段は Windows 使いだが、たまに Linux (Xubuntu) で遊ぶこともある。
日常的に使うツールや書き捨てのスクリプトはもっぱら Emacs Lisp で書くことが多い。
Python は大学の講義で学んだ程度。C・C++ は競技プログラミングで経験があり、Bash や PowerShell（cmd.exe）はコマンド操作や設定ファイルの変更・シェルスクリプト（バッチファイル）の記述のために、HTML5・CSS・JavaScript・PHP は本サイトの構築のために使用した。
その他スキルとしては Git・MySQL。
