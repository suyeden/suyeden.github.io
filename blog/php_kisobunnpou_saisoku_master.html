基本文法

コードブロック

PHPコードは開始タグ <?php から始まり、終了タグ ?> で終了します。
ただし、終了タグは省略することが推奨されています。

<?php
echo 'piyo';

文の終わり

PHPコードは、 ; で一文の終わりを示します。

echo 'hoge';

コメント

// 一行コメント
# 一行コメント
/*
複数行コメント
複数行コメント
*/

変数の宣言

型はPHPが自動で決定するので、宣言する必要はありません。

$a = 'hoge'; // string
$b = 1; // integer
$c = true; // boolean

数値の表現

四則演算

$i = 1 + 1; // 2
$i = 1 - 1; // 0
$i = 1 * 1; // 1
$i = 4 / 2; // 2
$i = 5 % 2; // 1

インクリメント・デクリメント

$i++;
$i--;

文字列操作

文字列はシングルクォートかダブルクォートで囲みます。

$s = 'hoge';
$s = "fuga";

特殊文字

ダブルクォートの中では特殊文字を利用できます。

$s = "hoge\nfuga"; // \nは改行
$s = 'hoge\nfuga'; // \nは文字列

変数展開

ダブルクォートの中では変数展開ができます。

$x = hoge;
$string1 = "$x fuga";
$string2 = "{$x} fuga";

結合

$string1 = 'hoge';
$string2 = 'fuga';
$s = $string1 . $string2; // hogefuga

配列

初期化

$array = [];

リスト

リストは、キーが数値の連想配列として表現されます。

$array = ['apple', 'melon', 'banana'];
echo $array[0]; // apple

配列の値を変数に入れることもできます。

$array = ['apple', 'melon', 'banana'];
[$x, $y, $z] = $array; // $x = 'apple', $y = 'melon', $z = 'banana'

連想配列

$array = [
  'apple' => 'red',
  'melon' => 'green',
  'banana' => 'yellow',
];
echo $array['banana']; // yellow

配列の最後の要素の後ろにカンマ,を置いてもいいです。
最後の要素の後にカンマを置くかどうかについては、時々論争が起きます。
カンマを置いていると、新しい要素の追加があった時に、カンマ忘れでエラーになりにくいですし、Git上での差分も少なくてすみます。

要素の追加

$array = ['apple', 'melon'];
$array[] = 'banana'; // $arrayは['apple', 'melon', 'banana']

連想配列の追加はキーを指定します。

$array = ['apple' => 'red', 'melon' => 'green'];
$array['banana'] = 'yellow'; // $arrayは['apple' => 'red', 'melon' => 'green', 'banana' => 'yellow']

特殊な演算子

エルビス演算子

$xがtrueであれば、$hogeにxを代入。そうでなければ、$hogeにyを代入する。

$hoge = $x ?: $y;

Null合体演算子（PHP7.0〜）

Nullの判定をする演算子です。

$hoge = $x ?? $y;

宇宙船演算子（PHP7.0〜）

左辺（a）と右辺（b）を比較し、aが大きい場合は 1、 aとb が等しい場合は 0、 bが大きい場合は -1 をそれぞれ返します。

switch (1 <=> 2) {
    case 1:
      echo '左辺の方が大きいです。';
      break;
    case -1:
      echo '右辺の方が大きいです。';
      break;
    case 0:
      echo '両辺は等しいです。';
      break;
}

定数

定数には配列も定義できます。定数は慣習的に大文字で書くことになっています。

define('ANIMALS', [
    'dog',
    'cat',
    'bird',
]);
echo ANIMALS[1]; // cat

制御文

if文

if ( 条件 ) {
    式
}

if-else文

if ( 条件 ) {
    式1
} else {
    式2
}

elseif文

if ( 条件1 ) {
    式1
} elseif ( 条件2 ) {
    式2
}

while文

条件がtrueの間、ループします。

while ( 条件 ) {
    式
}

for文

for ($i = 0; $i < 100; $i++) {
    式
}

foreach文

// 配列の値だけ取り出す場合
foreach ($array as $value) {
    式
}
// 配列のキーも取り出す場合
foreach ($array as $key => $value) {
    式
}

多次元配列の場合、複数の変数に値を渡すことができます。(PHP7.1〜)

$array = [
    ['hoge', 'hogehoge'],
    ['fuga', 'fugafuga'],
    ['piyo', 'piyopiyo'],
];
foreach ($array as [$single, $double]) {
    echo $single;
}
// hogefugapiyo

例外処理

例外を処理したい場合は、 try-catch 構文で処理します。

try {
    処理
} catch(Exception $e) {
    処理
}

関数

定義

return で返り値を返します。
呼び出し時に引数が立ちない場合、 ArgumentCountError の例外が発生します。（PHP7.1〜）

function sum($x, $y) {
    return $x + $y;
}
$total = sum(1, 2); // $total = 3

配列を使えば複数の値を返せます。

function plus_minus($x, $y) {
    return [$x + $y, $x - $y];
}
[$add, $sub] = plus_minus(10, 3); // $add = 13, $sub = 7

型宣言

引数と返り値にも型が指定ができる。

function hoge(string $s, int $x): bool {
    return true;
}

nullableな型も宣言できます。（PHP7.1〜）

function hoge(?string $s, ?int $x): ?bool {
    return null;
}

返り値がない場合、voidを指定することもできます。（PHP7.1〜）

function hoge(string $s): void {
    $GLOBAL['hoge'] = $s;
}

宣言可能な型

string
int
bool
array
float
callable
クラス名 / インターフェイス名
iterable （PHP7.1〜）
void （PHP7.1〜）
object （PHP7.2〜）

無名関数

関数に名前をつけずに関数を定義することができます。

$bar = 'world';
$hoge = function($foo) use ($bar) {
    var_dump($foo . ': ' . $bar);
};
$hoge('hello'); // string(12) "hello: world"

即時実行もできます。

(function($foo) use ($bar) {
    var_dump($foo . ': ' . $bar);
})('hello'); // string(12) "hello: world"

クラス

定義

アクセス修飾子はpublic、protected、privateが使えます。

class Human {
    public $age = 20;
    private $name = 'hoge';
    public function getName() {
        return $this->melon;
    }
}
$hoge = new Human();
echo $hoge->age; // 20
echo $hoge->getName(); // hoge
