Debian GNU/Linux

参考：https://toshio-web.com/debian10-buster-install

  1. Ubuntu 同様にインストール用メディアを用意し、これを起動する。
  2. 「Debian GNU/Linux installer menu」が立ち上がったら、「Graphical install」をクリック。
  3. 「Japanese - 日本語」を選択し、「continue」をクリック。
  4. 「日本」を選択し、「続ける」をクリック。
  5. 「日本語」を選択し、「続ける」をクリック。
  6. 「ホスト名」に任意の名前を入力し、「続ける」をクリック。（個人なら Debian で可）
  7. 「ドメイン名」に任意の値を入力して、「続ける」をクリック。（個人ならユーザー名と一緒で可。Webサーバーとして使うときは結局ドメイン取得する）
  8. rootユーザーのパスワードは未入力のまま、「続ける」をクリック。
    未入力の場合、このあと設定する初期ユーザーアカウントでsudoコマンドを実行すればroot権限を与えられる。（最近の主流？）
  9. 「ユーザーアカウント（フルネーム）」を入力して、「続ける」をクリック。（個人ならユーザー名と一緒で可）
  10. ユーザー名を入力して「続ける」をクリック。
  11. ユーザーパスワードを入力して、「続ける」をクリック。
  12. 「ガイド-ディスク全体を使う」を選択し、「続ける」をクリック。
  13. インストール先のディスクを選択して、「続ける」をクリック。
  14. 「すべてのファイルを1つのパーティションに（初心者ユーザには推奨）」を選択して、「続ける」をクリック。
  15. 「パーティショニングの終了とディスクへの変更の書き込み」を選択して、「続ける」をクリック。
  16. "ディスクのパーティショニング"にて、「ディスクに変更を書き込みますか？」に対して「はい」を選択して、「続ける」をクリック。
  17. インストールが行われるので、しばらく待つ。
  18. "パッケージマネージャの設定"にて、「別のCDやDVDを検査しますか？」に対して「いいえ」を選択して、「続ける」をクリック。
  19. "パッケージマネージャの設定"にて、「ネットワークミラーを使いますか？」に対して「はい」を選択して、「続ける」をクリック。
  20. 「日本」を選択して、「続ける」をクリック。
  21. 任意のアーカイブミラーを選択して、「続ける」をクリック。（deb.debian.org で大丈夫）
  22. プロキシ情報未入力のまま「続ける」をクリック。
  23. パッケージマネージャの設定が行われるので、しばらく待つ。
  24. 「Debian パッケージ利用調査に参加しますか？」に対しては「いいえ」を選択して、「続ける」をクリック。
  25. インストールするデスクトップ環境にチェックを入れ、「続ける」をクリック。
    （LXDE を選択。軽量性、カスタマイズ性（柔軟性）、安定性、デザイン（アイコン）等を考慮。）
  26. インストールが行われるので、しばらく待つ。
  27. 「マスターブートレコードに GRUB ブートローダをインストールしますか？」に対して「はい」を選択して、「続ける」をクリック。
  28. ブートローダのインストール先のディスクを選択して、「続ける」をクリック。
  29. "インストールの完了"にて「続ける」をクリック。
  30. 再起動後、ログインしてデスクトップ画面が表示されればインストールは完了。

Ubuntu

  1. 「Ubuntu Japanese Team (Japan Remix)」から isoファイル をダウンロード（保存）する。
  2. イメージファイル（isoファイル）をUSBメモリに書き込むためのソフトをダウンロードする。
    オススメ：Universal USB Installer
  3. イメージファイルをUSBメモリに書き込む。
    1. 「License Agreement」にて「I agree」をクリック
    2. 「OSの種類」から「Ubuntu」を選択
    3. 「Browse」ボタンをクリック
    4. ダウンロードしたisoファイルを選択し「開く」をクリック
    5. 「Select Your USB Flash Drive」にて書き込み先のUSBを選択し、「Create」ボタンを押すと書き込み開始
      ※ ここで間違ったドライブを選択すると一瞬にしてデータを失ってしまうため慎重に行うこと。必要に応じてバックアップは取っておくこと。
    6. USBに本当に書き込んでいいかの最終確認窓。間違いがない場合は「はい」をクリック
    7. 書き込みが完了したら「Close」ボタンをクリック
      インストール用メディアの完成
  4. Ubuntu を USB から起動する
    完成した USB で Ubuntu を仮利用することができる。
    パソコンに USB を挿して電源を入れ、同時に F2かF10かF12 を長押し（連打）する。
    ブートオプション（何を立ち上げるか選択できる画面）で USBメモリ を選択。
    Ubuntu 側の GNU Grub でも何を起動するか聞かれるため、「Try Ubuntu without installing」を選択。OS（Ubuntu）が立ち上がる。
  5. Ubuntu のインストール
    デスクトップ左上の「Ubuntu XXXX LTS のインストール」と書かれたアプリケーションを開く。
    指示に従って慎重に進めていく。

以下 Ubuntu を USB にインストールする場合

インストール用メディアとは別のUSBを用意する。
8GBでギリギリ。余裕を持って32GB以上のものを用意する。
上記 5 までは一緒。

  1. 「コンピュータ上には Ubuntu と Windows Boot Manager がインストールされています。どのようにしますか？」の画面にて、「それ以外」を選択。
  2. 「続ける」
  3. 自分が挿した USB を選択（容量の大きさで確認すること）。
    「変更ボタン」あるいは「ダブルクリック」でパーティションの作成（左下の「+」を押せたら押す？）。
  4. 「利用方法：ext 4 …」
    「マウントポイント：/（スラッシュ）」
  5. 「ブートローダをインストールするデバイス」にてインストール先にUSBメモリを選択（USBメモリから立ち上げるため、初期設定はPC内蔵のHDDのため注意）
  6. 「今すぐ再起動」を選択
  7. PC起動と同時に F2かF10かF12 を長押し（連打）、ブート選択画面で Ubuntu が表示されるため、これを選択して立ち上がればOK。
