Emmet記法 HTMLチートシート

記法 機能
！ HTMLのテンプレートを展開
div ＜div＞＜/div＞
a ＜a href=""＞＜/a＞・「a:」以降に"link"や"blank"等を付け加えられる
#id div#id名 → ＜div id="id名"＞＜/div＞
.class div.クラス名 → ＜div class="クラス名"＞＜/div＞
[id・クラス以外の属性] a[title="My Site"] → ＜a href="" title="My Site"＞＜/a＞
{テキスト} 要素にテキストを入れる
＊個数 同じ要素を複数作成
＄＊個数 連番の指定・＄の個数で連番の桁数を指定
＆@数字＊個数 開始数字を指定した上で連番の指定
＞ 階層(ネスト)の表現
＞要素1+要素2 同階層の要素の表現
要素1＞要素2要素3 要素3を要素1と同階層に戻す
(要素) 要素のグループ化 eg. (ul＞li＊3)＊2
form フォームタグの挿入
input:t インプットタイプテキストの挿入
input:r ラジオボタンの挿入
input:c チェックボックスの挿入
select セレクトボックスを挿入
lorem ダミー文章の挿入
img イメージタグの挿入

要素名でタグに展開できる
出てくる候補の右端の！マークでどのように展開されるかを見ることができる (VS Code)

div#reset.id.btn.disabled → ＜div id="reset" class="btn disabled"＞＜/div＞
div[data-id="5" title="My Site"] → ＜div data-id="5" title="My Site"＞＜/div＞
p{hello} → ＜p＞hello＜/p＞
div.item*10 → ＜div class="item"＞＜/div＞ が10個
div.item$$$*10 → ＜div class="item001"＞＜/div＞から＜div class="item010"＞＜/div＞まで連番で作成

暗黙的な補完機能

.container → ＜div class="container"＞＜/div＞
ul＞.item → ＜ul＞＜li class="item"＞＜/li＞＜/ul＞
table＞.row＞.col → ＜table＞＜tr class="row"＞＜td class="col"＞＜/td＞＜/tr＞＜/table＞

テキストを囲う

後からタブを上層に追加(選択範囲を別タブで囲う)する時に便利

  1. コマンドパレットを起動 (Ctrl + Shift + p)
  2. "＞wrap"と挿入し、"Wrap with Abbreviation"を選択
  3. emmet記法でタブを指定できる

選択範囲内の各行についてタブで囲いたい時は"Wrap Indivisual Lines with Abbreviation"を選択
eg.

- red
- blue
- pink

ul＞li*|t と挿入することで

<ul>
  <li>red</li>
  <li>blue</li>
  <li>pink</li>
</ul>

となる

画像に関する情報の取得

  • 画像サイズの取得
    カーソルがイメージタグにあることを確認してから、
    コマンドパレットを開いて ＞update と挿入して Update Image Size を選択する
    width と height が挿入される
  • 画像サイズの変更
    width="650" を width="650*2" とし、
    コマンドパレットを開いて ＞evaluate と挿入して Evaluate Math Expression を選択すると、
    width="1300" となっている
