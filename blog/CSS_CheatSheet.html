記述場所

・インライン
  <p style="font-size:20px;">

・エンベッド
  <style>
    p { font-size: 20px; }
  </style>

・リンク
  <link rel="stylesheet" href="style.css">

基本形

要素 { プロパティ: 値; }

#ID名 { プロパティ: 値; }

.クラス名 { プロパティ: 値; }

子孫セレクタ
親要素 子孫要素 { プロパティ: 値; }

複数セレクタ
セレクタ1, セレクタ2 { プロパティ: 値; }

全称セレクタ
 * { プロパティ: 値; }

疑似クラス
要素:last-child { プロパティ: 値; }
  疑似クラス名:
    :hover        マウスポインタが乗った
    :visited      リンクが訪問済み
    :first-child  最初の子要素
    :last-child   最後の子要素

疑似要素
要素::before { プロパティ: 値; }
要素::after { プロパティ: 値; }

文字のサイズ指定 (相対単位)

px : ブラウザ標準は 16px
%  : 親要素のサイズに対する%
em : 1文字文の倍数

プロパティ一覧

background-clip: 範囲の値
  背景の適応範囲を指定する
  範囲の値 : border-box | padding-box | content-box

background-color: 色
  背景色を指定する

background-image: url(画像ファイルのパス)
  背景画像を指定する
  url という文字列を忘れず入れること

background-origin: 基準位置
  背景の基準位置を指定する
  基準位置 : border-box | padding-box | content-box

background-position: 位置
  背景の表示開始位置を指定する
  位置 : top | bottom | left | right | center | %

background-repeat: 繰り返し方法
  背景の繰り返し方法を指定する
  繰り返し方法 : repeat-x | repeat-y | repeat | space | round | no-repeat

background-size: サイズ
  背景画像の表示サイズを指定する

background: 値
  背景に関する指定をまとめて行う
  background-color などの値を使用

  cf. 線形グラデーション
  background: linear-gradient(グラデーションの方向, #1つ目の色指定 かかり具合の距離%, #2つ目の色指定 かかり具合の距離%);
    グラデーションの方向:
      それぞれ上、下、左、右方向へ伸びる
        to top
        to bottom
        to left
        to right
      2つの向きを指定すると、四隅のいずれかの方向へ伸びる
        to top left
        to top right
        to bottom left
        to bottom right
      45degのように角度を度数で指定
        deg

  cf. 円形グラデーション
  background: radial-gradient(グラデーションの種類, #1つ目の色指定 かかり具合の距離%, #2つ目の色指定 かかり具合の距離%);
    グラデーションの種類:
      circle   正円
      ellipse  楕円

border-collapse: 重なり方
  表の隣接するセルの重なりを指定する
  重なり方 : collapse | separate

border-radius: 半径
  表の角を丸める
  2～4個の値を指定して4つの角を個別に変更することも可能
  角を個別に設定するための border-top-right-radius, border-bottom-right-radius, border-bottom-left-radius, border-top-left-radius もある

border: 線種 太さ 色
  ボーダー(枠線)を設定する
  四辺を別に設定するための border-top, border-bottom, border-left, border-right もある
  線種: none | solid | dotted | dashed | double | groove | ridge | inset

bottom: 距離
  要素の下からの位置を指定する
  positionプロパティと組み合わせて使用

box-shadow: Xシフト量 Yシフト量 ぼかし幅 色 inset
  要素に影を付ける
  insetを付けると要素の内側に影が付く

box-sizing: 計算方法
  ボックスモデルのルールを変更し、サイズの指定方法を変える
  計算方法 : content-box | padding-box | border-box

clear: クリア方法
  先行するフロートを解除する
  クリア方法 : left | right | both

color: 色
  文字の色を設定する

content: "文字"
  疑似要素と組み合わせ、要素の前後に文字を表示する

cursor: カーソル種類
  要素にマウスオーバーしたときのマウスポインタの種類を設定する
  カーソル種類 : pointer | text など

display: 表示特性
  要素の表示特性を切り替える
  表示特性 : none | inline | block | inline-block | table-cell など

float: フロート方法
  要素をフロート状態にする
  フロート方法 : left | right | none

font-family: フォント名
  文字のフォントを指定する
  カンマで区切って複数のフォントを列挙した場合、先に出現したそのパソコンで利用可能なフォントで表示される
  フォント名 : 任意のフォント名 | serif | sans-serif | monospace

font-size: サイズ
  文字のサイズを指定する
  smaller : 親要素を基準として1段階フォントサイズを小さくする(2割ほど小さくなる)

font-weight: 太さ
  文字の太さを指定する
  太さ : 100～900 | normal | bold | bolder | lighter

height: 高さ
  要素の高さを指定する

left: 距離
  要素の左からの位置を指定する
  positionプロパティと組み合わせて使用

letter-spacing: 字間
  字間を指定する

line-height: 行間
  行間を指定する

list-style: タイプ 位置
  リストの行頭アイコンの種類や位置を指定する
  タイプ : disc | circle | square | decimal

margin: 幅
  要素のマージン(ボーダーより外側の空き)を指定する
  2つの場合は上下,左右
  4つの場合は上,右,下,左
  2～4個の値を指定して4方向のマージンを個別に変更することも可能
  個別に設定するための margin-top, margin-bottom, margin-left, margin-right もある
  margin: ○ auto; -> 要素を左右中央揃えに

max-height: 高さ
  要素の最大の幅を指定する
  この指定値以上には広がらなくなる

max-width: 幅
  要素の最大の幅を指定する
  この指定値以上には広がらなくなる

min-height: 高さ
  要素の最小の高さを指定する
  この指定値以上には縮まなくなる

min-width: 幅
  要素の最小の幅を指定する
  この指定値以上には縮まなくなる

opacity: 不透明度
  要素の不透明度を0～1の数値で指定する

overflow: 表示方法
  要素からはみ出したコンテンツの表示方法を指定する
  表示方法 : visible | hidden | scroll | auto

padding: 幅
  要素のパディング(ボーダーより内側の空き)を指定する
  2つの場合は上下,左右
  4つの場合は上,右,下,左
  2～4個の値を指定して4方向のパディングを個別に変更することも可能
  個別に設定するための padding-top, padding-bottom, padding-left, padding-right もある

position: 配置方法
  ポジションでの配置方法を指定する
  top, bottom, left, right プロパティと組み合わせて使用する
  配置方法 : static | relative | absolute | fixed

right: 距離
  要素の右からの位置を指定する
  positionプロパティと組み合わせて使用

text-align: 配置方法
  行内でのテキストやインラインの要素の揃え方を指定する
  配置方法 : left | right | center | justify

text-decoration: 種類
  文字の下線や取り消し線を指定する
  種類 : none | underline | overline | line-through | blink

text-indent: 幅 hanging
  1行目のインデント(字下げ)幅を指定する
  hanging を付けるとぶら下げインデントになる

top: 距離
  要素の上からの位置を指定する
  positionプロパティと組み合わせて使用

transition: 対象プロパティ 変化の時間 変化の種類
  徐々に対象プロパティの値が変化するアニメーションにする
  変化の種類 :
    default         変化なし
    ease            開始と完了を滑らかにする
    linear          一定のスピード
    ease-in         ゆっくり始まる
    ease-ou         ゆっくり終わる
    ease-in-out     ゆっくり始まってゆっくり終わる
    cubic-bezier()  3次ベジエ曲線を表す4つの数字をカンマ区切りで指定

vertical-align: 基準
  インラインの要素の縦の揃え方を指定する
  基準 : baseline | text-top | text-bottom | middle | top | bottom

width: 幅
  要素の幅を指定する

@charset "utf-8";
