動画を結合する

以下のようにして複数ファイルを無劣化で結合できる

  1. インデックスファイルを作成する
    インデックスファイルには、下記のフォーマットで記入します。ファイル件数分改行して記載します。記述した順序でファイルが連結されます。

    file 'ファイル名1.mp4'
file 'ファイル名2.mp4'

    特に Windows では、動画ファイルとインデックスファイルは同階層におくこと。

  2. インデックスファイルを指定して無劣化コピーする

    ffmpeg (-safe 0) -f concat -i インデックスファイル.txt -c copy output.mp4

    あるいは以下のようにも記述できる。

    ffmpeg (-safe 0) -f concat -i インデックスファイル.txt -c:v copy -c:a copy -map 0:v -map 0:a output.mp4

    • オプション解説

      -safe 0 セーフモード解除（インデックスファイルのパスを絶対パスで指定しない場合は不要らしい）
      -f concat 結合する指示を出す
      -i files.txt インデックスファイルを指定
      -c copy オリジナルファイルのコーデックを再エンコードすることなく処理する
      -c:v copy ビデオコーデックをコピーする（無劣化）
      -c:a copy オーディオコーデックをコピーする（無劣化）
      -map 0:v ビデオファイルが複数存在する場合は全てのものを同梱する
      -map 0:a オーディオファイルが複数存在する場合は、全てのものを同梱する

      ※ -c copy
      高速に処理できるが、fileA.mp4、fileB.mp4のコーデックが異なる場合はこの方法では結合できない。
      異なるコーデックの動画を結合したい場合は、-c copyを外す（デフォルトのコーデックが使われる）かコーデックを指定してエンコードすること。

      ※ -safe 0
      「Unsafe file name '/path/to/a.mov' mylist.txt: Operation not permitted」というエラーが出て実行できない時に有効。

      動画を連結する場合は再エンコードしたほうが間違いないらしい。

秒数を指定して動画を切り出す

ffmpeg -ss [スタート秒数] -i [元ファイル] -t [長さ] -c copy [出力ファイル]

秒数は 00:00:00.000 （時間:分:秒.ミリ秒）のように記述する。

また以下のように記述することも可能。

ffmpeg -ss 120 -i src.mp4 -t 30 -c copy dst.mp4

上記の例だと 120 秒から 30 秒だけ切り出します。

-c copy によりトランスコードせずに出力しているため爆速です。
もし開始タイミングや動画の長さがズレる場合は -c copy オプションを外してトランスコードすると良くなるかと思います。
スタート秒数、長さは指定しなければ冒頭、末尾になります。

映像と音声を結合する

ffmpeg -i video.mp4 -i audio.wav -c:v copy -c:a aac -map 0:v:0 -map 1:a:0 output.mp4

ポイントは-c:v copyを指定して動画を再エンコードしないようにしていることと、-map 0:v:0 -map 1:a:0を指定して元の動画ファイルに音声トラックが含まれている場合でも確実に音声ファイルの音声トラックを利用するようにしているところです。
元の動画ファイルに音声が含まれていないのであれば、-mapの指定はなくても良いです。
