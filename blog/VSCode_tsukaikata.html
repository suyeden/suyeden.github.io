キーボードから手を離さずにVS Codeで開発

ショートカットキー 機能
Alt ツールバーにアクセスできる
Ctrl + Shift + P コマンドパレットが開く
Ctrl + Shift + E ファイルエクスプローラーを探索
Ctrl + Q 各領域に移動
Ctrl + B サイドバーの開閉
Ctrl + @ ターミナルを開く・Ctrl + Shift + @で新しいターミナル・Ctrl + @で閉じる
Ctrl + Tab 開いているタブから選んで移動できる・Shiftまぜれば逆順に移動
Ctrl + 1 エディタが分割されているとき、最初の分割にフォーカスする
Ctrl + W タブを閉じる
Ctrl + Z Undo
Ctrl + Shift + Z Redo
Ctrl + Shift + \ 対応するカッコに飛ぶ
Ctrl + Click リンク先に飛ぶ

VS Code 使用の流れ

  • ファイルの編集
    アクティビティーバーからエクスプローラーを開き、フォルダーを追加する。
    エクスプローラー画面で新規ファイルやフォルダーを作成できる。
    右クリックでセーブやリネーム、新規ファイルやフォルダーの作成もできる。
  • Emmet記法
    ! Return : HTMLのテンプレートを呼び出す
    ul>li*3 : <ul>直下に<li>を3つ作成する
    等といった独自の記法を利用できる
  • css, javascript の読み込み
    css:src でテンプレートを挿入できる。
    作成したリンクにマウスホバーしてCtrl+clickすることでファイルをopen/createできる
    マウスでタブをドラッグすることで画面分割もできる
  • CSSの編集
    色を書くと見本が出てき、マウスホバーするとカラーピッカーも出てくる。
    セレクターにマウスホバーすると要素を例示してくれる。
  • 設定の変更
    Ctrl+, で設定画面を開ける。
  • コマンドパレットを使う
    Ctrl + Shift + P で起動、ESCで抜ける
    theme と入力することでカラーテーマが選択できたり、
    minimap と入力することでミニマップ関連の設定を行える。
    ショートカットキーの検索にも使える。
    shortcut と入力して キーボード ショートカットの設定を開ける
    検索して様々な操作を行える。
  • マルチカーソルを使う
    クリックした箇所にカーソルを増やす → Alt + Click
    同じ単語を選択する → Ctrl + D
    同じ単語をいっきに選択する → Ctrl + Shift + L
    カーソルを垂直方向に増やす → Ctrl + Alt + 上下矢印

サイドバーの表示／非表示

例えば［エクスプローラー］ビューがサイドバーに表示されている状態で、これに対応するアイコンをアクティビティーバーでクリックするとサイドバーが非表示になる。

エクスプローラー

[開いているエディター]
ユーザーが現在開いているエディタ（タブ）がエディタグループごとに表示される。

[フォルダ名]
VS Codeで現在オープンしているフォルダを表す。
その下には該当フォルダにあるフォルダやファイルが表示されている。
１度につき開けるフォルダは基本１ルートフォルダまで。

　例外 : マルチルートワークスペース

　以下５通りの方法
　　・［ファイル］メニューの［ワークスペースにフォルダーを追加］を選択する
　　・［エクスプローラー］ビューを右クリックしてコンテキストメニューから［ワークスペースにフォルダーを追加］を選択する
　　・コマンドパレットで「workspace」などと入力して［ワークスペース：ワークスペースにフォルダーを追加］を選択してから追加するフォルダを指定する
　　・Windowsエクスプローラーから直接フォルダを［エクスプローラー］ビューにドラッグ＆ドロップする
　　・コマンドラインから「code -add」に続けてワークスペースに追加するフォルダを複数指定する

　マルチルートワークスペースからフォルダを削除するには、削除したいフォルダを［エクスプローラー］ビューで右クリックしてコンテキストメニューから［ワークスペースからフォルダーを削除］を選択。
　マルチルートワークスペースにすると、［ファイル］メニューの［フォルダーを閉じる］が［ワークスペースを閉じる］に変わる。

エディタ

1つ1つのタブのことを「エディタ」と呼び、それらを論理的なグループにまとめたもののことを「エディタグループ」と呼ぶ。
［エクスプローラー］ビューで「シングルクリック」すると、そのファイルがプレビューエディタに表示される。
［エクスプローラー］ビューでファイルをダブルクリックするか、プレビューエディタのタブをダブルクリックすると、それが通常のエディタに表示される。
プレビューエディタのタブにはイタリック体でファイル名が、通常のエディタのタブには通常書体でファイル名が表示される。

クイックオープンとコマンドパレット

クイックオープン

キーボードで［Ctrl］＋［P］キーを押すと、VS Codeの上部にクイックオープンウィンドウが表示される。
ここからはファイルを開いたり、指定した番号の行に移動したり、エディタグループを切り替えたり、何らかのコマンドを実行したりすることが可能。キーボード主体でVS Codeを使いたければ覚えておこう。
開きたいファイルにカーソルを合わせて［Enter］キーを押すと、そのファイルが開かれる。さらに［→］キーを押すと、クイックオープンウィンドウが開いたまま、バックグラウンドでそのファイルが開かれる。複数のファイルをクイックオープンウィンドウからまとめて開くときには便利。

コマンドパレット

キーボードで［Ctrl］＋［Shift］＋［P］キーを押すとVS Codeの上部に「コマンドパレット」が表示される（実際には上で紹介したクイックオープンウィンドウで「>」を入力してもよい）。
VS Codeで実行できる各種コマンドには名前が付いていて、その名前をこのコマンドパレットに入力することでそれらを実行できる。

文字コード／行末コード／ファイルの種類を選択する

ステータスバーの右下には、現在編集中のファイルの文字コードが表示される。ここをマウスでクリックすると、上記のコマンドパレットと同じ位置に「アクションの選択」と書かれたボックスが表示される。選択できるのは［エンコード付きで再度開く］（Reopen with Encoding）と［エンコード付きで保存］（Save with Encoding）だ。カーソルキーを使って、いずれかを選択するか、英字表記にマッチするように文字列を入力して［Enter］キーを押す。どちらを選択した場合も続いて文字コードを指定するドロップダウンが表示されるので、文字コードを選択すると、指定した文字コードでのファイルの再オープンもしくはファイルの保存が行われる。
