ブランク・サスペンド・ハイバネート・シャットダウン・ディスプレイの電源を切る・画面をロック・スリープ について

ディスプレイのみについて言っているのか、作業内容のメモリへの保存等ディスプレイ以外の要素も含めているのか、に注目する必要がある。
ブランクは、ディスプレイの電源を切ったり、画面を黒一色にしたり、意味はケースバイケース。画面をロックするかどうかは設定次第。
サスペンドはスリープのこと。起動中のPCの状態を保持したままハードウェアへの給電を抑える。
ハイバネートは休止状態のこと。起動中のPCの状態をストレージに保存して電源を「オフ」にし、次回PC起動時に保存した状態をストレージから復帰させる。

apt

# パッケージのインストール
sudo apt install [パッケージ名]
# パッケージの削除
sudo apt remove [パッケージ名]
# パッケージの完全削除（環境設定まですべて削除）
sudo apt purge [パッケージ名]
# 今現在使用されていない（依存関係がない）パッケージの削除
sudo apt autoremove
# パッケージの検索
sudo apt search [パッケージ名]
# パッケージの一覧表示
sudo apt list --installed
# アップグレード可能なパッケージを一覧表示
sudo apt list --upgrade
# 確認無しで apt を実行
sudo apt -y ...

apt と apt-get の違い

apt-get は最初のコマンドラインベースフロントエンド、apt は apt-get の持っていた設計上のミスを克服。
Ubuntu では apt コマンドの使用が推奨されている。

sudo と su

sudo

  • root のパスワードでなくてユーザーのパスワードを必要とする
  • sudo を許す許さないは管理者 root が決める
  • sudo でやったこと、やろうとして失敗したことは、すべてログに残る

su

  • root のパスワードを知っている必要がある
  • root のパスワードを知っている人が su することを禁じることはできない
  • su 後はログが残らず、やりたい放題

Ubuntu と root

Ubuntu は初期状態では root が使えない（パスワード未設定）ようになっている
su を実行するには以下のようにする

sudo su
