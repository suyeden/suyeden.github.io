置換操作

M-%
スペースキーか'y'を押せば置換が行われ、'n'を押すと置換は行わず、次の候補に移動します。
終了するときには、Enterキーか'q'を入力します。上に表示されている通り '?' を入力すればコマンドのリストが表示されます。

確認なしに一括して行いたいとき
M-x replace-string
置換のための文字列を入力すれば、一括置換が行われます。

local-set-key

キーバインドに矢印キーを用いる時や、ある特定のモードが有効になったタイミングで設定したい時

(local-set-key (kbd "キーバインド (eg. C-c C-x)") '関数名)

特定キーマップへのキーバインドの設定

;;; keymap
(defvar org-mode-map (make-keymap))
;;; define-key
(define-key org-mode-map "\C-cn" 'new-blog)
(define-key org-mode-map "\C-ce" 'end-blog)
(define-key org-mode-map "\C-c\C-h" 'blog-help)
(define-key org-mode-map "\C-cx" 'all-export-to-html)
(define-key org-mode-map "\C-c\M-s" 'blog-insert-space)

キーマップの継承

キーマップkeymapの親キーマップとしてparentを設定する。

(defvar parent (make-keymap))
(defvar keymap (make-keymap))
(set-keymap-parent keymap parent)

dired

diredはディレクトリ一覧をバッファに作成し、一覧中にあるファイルを操作できます。

キーバインド コマンド内容
C-x d diredの起動
C-x 4 d diredを別のウィンドウで起動
d このファイルに削除フラグを付ける
u この行の削除フラグを消す
x 削除フラグが付いたファイルを（本当に）削除する
＃ （名前が＃で始まり＃で終る） すべての自動保存ファイルに削除フラグを付ける
~ （名前が~で終る）すべてのバックアップファイルに削除フラグを付ける
f or Return C-x C-fと打鍵したのと同様に, 現在行が表すファイルを訪問する
o fと同様だが, ファイルのバッファを別のウィンドウに表示する
C-o 現在行が表すファイルを訪問して別のウィンドウに表示するが, そのウィンドウを選択しない

変更内容による Emacs キーバインディングの設定方法

  • 既存のコマンドを引数を与えないで割り当てる -> そのまま

    (define-key global-map "\C-ca" 'fill-region)

  • 既成のコマンドに引数を与えて割り付ける -> 説明が必要 (ラムダ式で十分)

    (define-key global-map "\C-ca" 
  '(lambda ()
     (interactive)
     (backward-delete-char 2)))

  • 自作のコマンドを作成して割り付ける -> 説明が必要 (コマンドとは何かなど)

    (define-key global-map "\C-ca" 'my-command)
;;
(defun my-command ()
  (interactive)
  (insert "a"))

メジャーモードとマイナーモード

メジャーモードは一つのバッファに対して同時に一つしか設定できない。
一方マイナーモードはメジャーモードとも他のマイナーモードとも併用することができる。
文章を自動的に折り返す auto-fill-mode や overwrite mode 等が代表的な副モードである。

defadvice

定義済みの関数に処理を付け足すことができる。

(defadvice 定義済み関数 (class 自分で決めた仮の関数名)
  処理)
;; 処理を付け足した関数を有効にする
(ad-activate '定義済み関数)

class では、 after か before (か around) を指定する。

OSの違いを判定する

変数 system-type を用いる。
